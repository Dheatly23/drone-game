# Drone Game

## What is this?

Well, it's a demo of WebAssembly integration in a game.
Right now, it's an incomplete prototype.
But soon it might get done.

## Status

Still in progress.

## How to build

### Dependecies

- Rust
- `cargo-make`

### Steps

1. Install [godot_wasm](https://github.com/Dheatly23/godot-wasm).
2. Run `run.bat` (Windows) or `run.bash` (Linux).
